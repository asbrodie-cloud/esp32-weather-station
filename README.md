# esp32-weather-station

This project implements a weather monitoring system using an ESP32.

## Features
- Temperature & humidity (DHT22)
- Pressure & altitude (BMP280)
- Soil moisture sensing
- TFT display dashboard
- Web-based remote monitoring

## Hardware Used
- ESP32 Dev Board
- DHT22 Sensor
- BMP280 Sensor
- Soil Moisture Sensor
- 2.8" TFT Display

## Communication
- I2C (BMP280)
- SPI (TFT Display)
- Analog (Soil Sensor)
- Wi-Fi (Web Interface)
